# fluidity. Templates Reference

## Comments

Regular comment:
```html
<h2 class="text-2xl">Name</h2>
<p>Details</p>
````

Artist/Owner comment (verified):

```html
<h2 class="text-2xl">Name &check;</h2>
<p>Details</p>
```
